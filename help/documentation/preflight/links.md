# プリフライトリンクの機会

![ プリフライトリンクの商談 ](./assets/links/hero.png){align="center"}

Sites Optimizerのプレフライトリンクのオポチュニティは、web サイトでのリンクの使用に関連する問題を特定し、解決するのに役立ちます。 このオポチュニティは、サイトのリンク構造の評価に焦点を当て、改善のための実用的な推奨事項を提供します。
