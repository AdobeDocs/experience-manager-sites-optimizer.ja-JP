# プリフライトメタデータの商談

![ プリフライトメタデータの商談 ](./assets/metadata/hero.png){align="center"}

Sites Optimizerのプリフライトメタデータのオポチュニティは、web サイトでのメタデータの使用に関連する問題を特定し、解決するのに役立ちます。 このオポチュニティは、サイトのメタデータ構造の評価に焦点を当て、改善のための実用的な推奨事項を提供します。
