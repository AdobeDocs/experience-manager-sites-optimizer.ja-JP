# プリフライトでの監査結果

監査が完了すると、プリフライトには監査結果が商談として表示されます。 各商談はタイプ別に整理され、ページの強化と最適化に役立つレコメンデーションが含まれています。

AEMのプリフライトダイアログの上部には、全体的な監査結果を反映したユーザーの進行状況バーがあります。 問題なしで合格した商談の割合と、すべての商談で見つかった問題の合計数が表示されます。 ユーザーの進捗状況バーを使用して、ページ全体の健全性を一目で確認できます。 バーは色分けされています。

* 商談の&#x200B;**1/3**&#x200B;未満が完了しました
* **1/3 ～ 2/3完了**&#x200B;のオレンジ
* **2/3より多い完了**&#x200B;の緑
* 監査がまだ&#x200B;**実行中**&#x200B;の間は青

使用可能な商談タイプの完全なリストと、それらの対応方法を参照してください](./overview.md#preflight-opportunities)。[
