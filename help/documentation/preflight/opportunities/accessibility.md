# プリフライトのアクセシビリティオポチュニティ

![プリフライトのアクセシビリティの機会](./assets/accessibility/hero.png){align="center"}

Sites Optimizerのプリフライトアクセシビリティオポチュニティは、web サイト上のアクセシビリティの問題を特定および対処し、障害のあるユーザーを含むすべてのユーザーが使用できるようにするのに役立ちます。 この機会では、サイトのアクセシビリティ標準へのコンプライアンスの評価に焦点を当て、改善のための実用的なレコメンデーションを提供します。
