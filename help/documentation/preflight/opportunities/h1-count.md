# プリフライトの H1 カウントの機会

![プリフライトの H1 カウントの機会](./assets/h1-count/hero.png){align="center"}

Sites Optimizer のプリフライト（H1 カウント）オポチュニティは、web サイトでの H1 見出しの使用に関連する問題を特定して対処するのに役立ちます。 この機会では、サイトの構造の評価に焦点を当て、改善のための実用的なレコメンデーションを提供します。
