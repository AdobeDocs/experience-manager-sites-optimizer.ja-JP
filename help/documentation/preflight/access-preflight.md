# プリフライトにアクセス

プリフライトは、ユニバーサルエディター、ドキュメントベースのオーサリング、AEM Sites ページエディターなど、AEM Sitesのオーサリング環境で使用できます。 プリフライトの開封数は、オーサリング中のページのパネルで、[そのページで監査](./audits.md)を実行し、[公開前にページを強化するために見つかった商談](./audit-results.md)を表示します。

* **ユニバーサルエディター** - プリフライト拡張機能が&#x200B;**サイドパネル**&#x200B;に表示されます。 このオプションを選択すると、現在のページの監査が開始されます。
* **ドキュメントベースのオーサリング** - Sidekickまたはブックマークレットを使用して、プレビュー済みのページコンテンツでプリフライトツールを実行し、オポチュニティのリストを表示します。
* **AEM Sites ページエディター** - ブラウザーのプリフライトブックマークレットを使用して、監査を開始します。

セットアップ手順については、[&#x200B; プリフライト設定](./setup.md)を参照してください。
