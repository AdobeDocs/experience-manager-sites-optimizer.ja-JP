# プリフライトでの監査

Preflightはページを監査し、公開前にコンテンツを強化する機会を特定します。 ページに対してプリフライト監査を実行するには：

1. [ オーサリング環境](./access-preflight.md) （ユニバーサルエディター、ドキュメントベースのオーサリング、またはAEM Sites ページエディター）で監査するページを開きます。
1. [ プリフライトパネル ](./access-preflight.md)を開きます。
1. プリフライトは、エディターで開いているページに対するすべての監査を自動的に実行し、見つかった機会を表示します。
